In this text file you will find a snapshot of a complete Docker / Sharelatexserver installation for Ubuntu-Linux systems. This installation protocol refers to elementary os. This Linux system looks very elegant and is very similar to Mac OSX.

Elementary OS is a beautiful Linuxsystem which looks like Mac OSX.

Snap is a packagemanager which installs Software on Ubuntu systems.
Snap can be installed on elementary OS (or any Ubuntu based Linux systems)
from the command line. 

Open Terminal from the Applications launcher and type the following:

sudo apt update 

sudo apt install snapd

Next, make sure we set the correct PATH environment. Continue with this command:

PATH=/snap/bin:$PATH

sudo snap install docker
There is more to do, here comes the detailed protocol...


To run a command as administrator (user "root"), use "sudo <command>".
See "man sudo_root" for details.
